# 证件管理程序

一个Python命令行程序，用于管理企业/组织人员证件有效期。自动筛选出证件即将过期的人员，并输出醒目的CSV报告。

## 功能特点

- 🔍 **智能列名识别**：自动识别常见的Excel列名格式
- 📊 **分类预警**：
  - 🔴 **紧急**：剩余天数 < 30天
  - 🟡 **预警**：剩余天数 30-90天
  - 🟢 **正常**：剩余天数 ≥ 90天
- 📝 **醒目报告**：输出带emoji标识的CSV报告，便于快速识别
- 🌐 **中文支持**：完美支持中文，使用UTF-8-sig编码

## 安装

### 1. 克隆或下载项目

```bash
cd certificate-manager
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 基本用法

```bash
python certificate_checker.py input.xlsx output.csv
```

### 参数说明

- `input.xlsx`：输入的Excel文件路径（必需）
- `output.csv`：输出的CSV报告路径（必需）

## Excel文件格式要求

### 必需列

| 列类型 | 支持的列名 |
|--------|-----------|
| 姓名 | 姓名、名字、Name、name、员工姓名、人员姓名 |
| 证件类型 | 证件类型、证书类型、Certificate Type、证件名称、证书名称 |
| 有效期 | 有效期、到期日、有效期至、expiry_date、expire_date、有效日期、证件有效期、证书有效期 |

### 可选列

| 列类型 | 支持的列名 |
|--------|-----------|
| 联系方式 | 联系方式、电话、手机、Phone、Tel、联系电话、手机号 |

### 支持的日期格式

- `2024-03-15` / `2024/03/15` / `2024.03.15`
- `15-03-2024` / `15/03/2024` / `15.03.2024`
- `2024年03月15日`
- Excel日期格式

## 技术栈

- Python 3.7+
- pandas - 数据处理
- openpyxl - Excel文件读取

## 注意事项

1. 确保Excel文件包含必需的列（姓名、证件类型、有效期）
2. 日期列应使用可识别的日期格式
3. 输出的CSV文件使用UTF-8-sig编码，可直接用Excel打开
4. 已过期的证件会显示负数剩余天数，归入"紧急"类别
